Wireshark tool

Wireshark - Wireshark proqramı təhlükəsizlik və şəbəkə kimi müxtəlif sahələrdə istifadə olunan açıq mənbəli və pulsuz şəbəkə trafikinə nəzarət proqramıdır. O, şəbəkə təlimi, təhlil, rabitə protokolunun inkişafı və şəbəkə problemlərinin aradan qaldırılması kimi müxtəlif məqsədlər üçün istifadə olunur. Bu proqram Windows, Linux və Unix daxil olmaqla müxtəlif platformalarda işləyir. Siz Wireshark proqramının köməyi ilə şəbəkə paketlərini tuta və onları dənəvər səviyyədə göstərə bilərsiniz. Bu alət sizə şəbəkə trafikinizi yoxlamağa və süzgəcdən keçirməyə, problemlərin əsas səbəbini böyütməyə, şəbəkə təhlilinə və nəticədə şəbəkə təhlükəsizliyinə kömək etməyə imkan verir.
Wireshark 1998-ci ildə Gerald Combs tərəfindən yaradılmışdır və şəbəkə mütəxəssisləri və proqram təminatı tərtibatçılarından ibarət qlobal təşkilatdır. Wiresharkvə onu yeni şəbəkə texnologiyaları və şifrələmə üsulları ilə yeniləyir.

Wareshark-ın quraşdırılması:

İlk öncə terminal üzərindən Wareshark-ı quraşdırmaq üçün aşağdakı addımları ardıcıl edirəm.

wireshark -h əmrini verdikdə Wireshark proqramını yoxlaya və sistemdəki şəbəkə trafikini təhlil edə bilirik.

Şəbəkə protokolu analizatoru:

Şəbəkə protokolu analizatoru - bir şəbəkədən keçən trafikin interaktiv bir şəkildə, real vaxt rejimində tutulmasına və analiz edilməsinə imkan verir.

  • Siyahıda çəkmək istədiyim interfeysi seçirəm.
    Ümumiyyətlə, əgər kompüteriniz Ethernet vasitəsilə qoşulubsa, bu “eth0” və ya Wi-Fi bağlantısından istifadə edirsinizsə, “wlan0” olacaqdır.
  • Üst paneldəki ilk işarəni vururam.
    Həmçinin ana səhifədəki interfeys adına iki dəfə klikləyə, çəkmə menyusundan istifadə edə və ya sadəcə CTRL+E düymələrini sıxa bilərəm.

Paket təhlili:

Bu zaman ekran üç əsas hissəyə bölünür:

  • Paket siyahısı: Burda mənbə, təyinat və protokol kimi makro məlumatı görünür. Məsələn, müəyyən bir IP ünvanından şübhəli HTTP fəaliyyəti axtarıramsa, əlaqəsi olmayan hər şeyi (DNS sorğuları və digər IP ünvanları kimi) atlaya bilərəm.
  • Paket təfərrüatları: Bir paket seçilən zaman onun məzmununu az və ya çox oxunaqlı mətn formatında görünür.
  • Paket baytları: Bayt və onaltılıq formatlı dəqiq paket məzmunu görünür.

Filtrlər

Ancaq Wireshark-da xüsusi bir şey axtarılan zaman əsas məsələ paketlər siyahısını analizdən keçirməkdir. Buna görə də Wireshark ekranın yuxarısında filtlər yaxın bir sahəni ehtiva edir, burada yalnız potensial olaraq maraqlı olan paketləri göstərmək (və ya onları istisna etmək) üçün düstur daxil edə bilinir.

Təkrar ötürmə problemi

Wireshark tez-tez daha mürəkkəb şəbəkə problemlərini müəyyən etmək üçün istifadə olunur. Şəbəkə çoxlu təkrar ötürmələrlə qarşılaşırsa, sıxlıq yarana bilər.

Rəng kodlaması

Wireshark ümumi mənada rəng kodlamasını tətbiq etməklə paket növlərini müəyyən etməyə kömək etməyə çalışır. Aşağıdakı cədvəl əsas paket növlərinə verilən standart rəngləri təsvir edir.

Wireshark-da rəng Paket Tipi
Açıq bənövşəyi TCP
Açıq mavi UDP
Qara Səhvləri olan paketlər
Açıq yaşıl HTTP trafiki
Açıq-sarı Server Mesaj Blokları (SMB) və NetBIOS daxil olmaqla Windows-a xüsusi trafik
Tünd sarı Marşrutlaşdırma
Tünd Boz TCP SYN, FIN və ACK trafiki

Defolt rəngləmə sxemi aşağıdakı kimidir.
